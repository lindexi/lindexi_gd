## 四、沙盒环境

### 4.1 沙盒设计

| 维度 | 沙盒配置 |
|---|---|
| **测试数据隔离** | 所有测试数据集存放于独立分区/目录，禁止写入原图目录 |
| **配置文件隔离** | 测试用配置存放于 `~/test-sandbox/config/`，不污染用户配置 |
| **输出隔离** | 截图/录屏/日志统一存放于 `~/test-sandbox/output/` |
| **进程隔离** | 测试运行时的进程 PID 单独记录，便于异常定位 |

### 5.1 离线配置策略

| 层级 | 配置方式 |
|---|---|
| **物理层** | 拔除网线 / 关闭 Wi-Fi |
| **系统层** | Windows：禁用所有网络适配器 / Linux：`nmcli networking off` |
| **防火墙层** | 出站规则全 deny（仅本地回环允许） |
| **应用层** | Wireshark 抓包验证（兜底确认） |

### 5.2 离线验证步骤

```bash
# Step 1: 物理层断开
nmcli networking off                          # Linux
Disable-NetAdapter -Name "Wi-Fi" -Confirm:$false   # Windows

# Step 2: 启动 Wireshark 抓包
wireshark -i any -k -w ~/test-sandbox/output/logs/offline-verify.pcapng

# Step 3: 启动图片查看器，打开 10 张图片，进行 5 分钟操作
# （开图/翻看/缩放/旋转/全屏）

# Step 4: 停止抓包，检查是否有非本地回环请求
wireshark -r ~/test-sandbox/output/logs/offline-verify.pcapng \
  -Y "ip.dst != 127.0.0.1 && ip.dst != ::1" \
  -T fields -e ip.dst | sort -u | wc -l
# 预期输出：0 ✅
```

### 6.3 冒烟脚本（伪代码）

```python
# 文件：~/test-sandbox/smoke-test.py
# 用途：5 测试机冒烟测试
# 依赖：pywinauto (Windows) / dogtail (Linux)

import time
import sys

ACTIONS = [
    ("A1_打开图片", "AC-01", "sample.jpg", 1.5),
    ("A2_翻看下一张", "AC-04a", "→", 1.0),
    ("A3_缩放", "AC-03", "Ctrl+=", 0.5),
    ("A4_旋转", "AC-05", "Ctrl+,", 0.5),
    ("A5_全屏", "AC-06", "F11", 0.5),
]

def smoke_test(platform):
    print(f"=== {platform} 冒烟测试开始 ===")
    results = []
    for action, ac, key, timeout in ACTIONS:
        t0 = time.time()
        # ... 实际操作 ...
        elapsed = time.time() - t0
        passed = elapsed <= timeout
        results.append((action, ac, elapsed, timeout, passed))
        print(f"  [{action}] AC={ac} key={key} 用时={elapsed:.2f}s / 阈值={timeout}s → {'✅' if passed else '❌'}")
    return results

if __name__ == "__main__":
    platform = sys.argv[1] if len(sys.argv) > 1 else "unknown"
    results = smoke_test(platform)
    failed = [r for r in results if not r[4]]
    if failed:
        print(f"\n❌ 冒烟测试失败：{len(failed)} 项不达标")
        sys.exit(1)
    print(f"\n✅ 冒烟测试通过：5/5 项达标")
```
