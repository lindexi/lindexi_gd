### 4.3 用例遗漏点识别

| # | 遗漏点 | 风险 | 建议 |

| **R-T2** | 极小窗口（如 200×150）下的布局 | 工具栏按钮被截断 | 建议增加窗口尺寸边界用例 |
| **R-T3** | 1KB 以下小图在 10% 缩放下的可见性 | 可能不可见导致误判 | 建议增加小图 + 极小缩放组合用例 |
| **R-T4** | 多显示器切换（DPI 不同） | 拖到外接屏时 UI 错位 | 建议增加跨屏用例 |
| **R-T5** | 长时间运行（如 8 小时）的内存稳定性 | 内存泄漏 | 建议增加 24 小时稳定性测试（放入探索性测试）|
| **R-T6** | 损坏图片在多平台的一致性表现 | Linux 与 Windows 错误信息不同 | 建议增加跨平台错误一致性用例 |
| **R-T7** | SVG 矢量图无上限缩放的性能曲线 | 缩放越大越卡 | 建议增加 SVG 性能专项用例 |
