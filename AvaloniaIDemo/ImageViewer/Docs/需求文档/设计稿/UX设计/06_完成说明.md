## 关键设计决策总结

| # | 决策 | 简要理由 |
|---|------|---------|
| DD-1 | 顶部工具栏 + 底部状态栏，不可隐藏 | 板正气质，不搞"悬浮"或"自动隐藏" |
| DD-2 | 工具栏按功能分组（文件/缩放/旋转/视图/更多） | 逻辑清晰，减少认知负担 |
| DD-3 | 缩放以鼠标位置为中心 | Photoshop/浏览器标准行为 |
| DD-4 | 旋转非破坏性，关闭时询问 | DD-6 容错性原则 |
| DD-5 | 全屏工具栏 1.5 秒渐隐，鼠标移动浮现 | 沉浸看图的同时保持操作可达 |
| DD-6 | 单实例替换，静默丢弃前一张的旋转/缩放状态 | 简单明确，不干扰用户新操作 |
| DD-7 | v1.0 快捷键不可自定义 | 板正克制，设定即最终 |
| DD-8 | 全屏下滚轮=翻看（非缩放），Ctrl+滚轮=缩放 | 全屏看图的自然行为是"翻看" |
