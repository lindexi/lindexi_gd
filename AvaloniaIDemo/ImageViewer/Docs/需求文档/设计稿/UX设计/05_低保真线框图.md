# 05 — 低保真线框图

> **文档定位**：用 ASCII 线框描述关键状态界面骨架，标注每个元素的优先级、状态、交互行为。  
> **关联 AC**：全部  
> **说明**：以下为低保真示意，用于指导 UI 设计师产出高保真设计稿。

---

## 状态索引

| 编号 | 状态 | 触发条件 |
|------|------|---------|
| S1 | 默认浏览状态 | 打开一张支持格式的图片 |
| S2 | 空状态 | 应用启动但未打开任何图片 |
| S3 | 缩放状态 | 用户放大/缩小图片后 |
| S4 | 旋转状态 | 用户旋转图片后 |
| S5 | 全屏状态 | 进入全屏模式 |
| S6 | 全屏·工具栏浮现 | 全屏模式下鼠标移动 |
| S7 | 错误·不支持格式 | 打开不支持的格式 |
| S8 | 错误·文件损坏 | 打开损坏的图片文件 |
| S9 | 加载中 | 图片正在加载（慢速存储） |
| S10 | 单实例替换 | 已有窗口，新打开另一张图片 |

```
优先级标注：
  P0 = 核心内容，始终可见

  - 状态栏缩放比例：当前100%时不显示（或无特殊标记）

```

## S10 — 单实例替换

```
┌──────────────────────────────────────────────────────────────┐
│ [Icon]  new_photo.png - 图片查看器          [_] [□] [X]     │  ← 标题栏更新为新文件名
├──────────────────────────────────────────────────────────────┤
│ [📂打开] │ [-] [100%▼] [+] [1:1] │ [↺] [↻] │ [⊞全屏] │ [⋮] │  ← 缩放重置
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                                                              │
│                    ┌─────────────────┐                       │
│                    │                 │                       │
│                    │   新图片内容    │                       │  ← 替换为新图片
│                    │   (适应窗口)    │                       │
│                    │                 │                       │
│                    └─────────────────┘                       │
│                                                              │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│ ● 第 1/1 张  │  new_photo.png    │  2560×1440  │  5.1MB     │  ← 状态栏更新
└──────────────────────────────────────────────────────────────┘

触发前条件：已有一个窗口显示 photo_2024.jpg（旋转90°，缩放150%）
新打开：new_photo.png

替换行为：
  1. 窗口置顶/聚焦
  2. 标题栏文件名更新
  3. 图片内容替换
  4. 旋转角度重置为 0°
  5. 缩放置为"适应窗口"
  6. 状态栏全部更新
  7. ✅ 不弹出旋转保存询问（静默丢弃上一张的旋转状态）
     ⚠️ 这是 UX 设计决策：单实例替换时，上一张的编辑状态不保留
     如果需求希望保留，请需求分析师确认
```

## 与 UI 设计师的协作要点

| UX 线框元素 | UI 设计师细化方向 |
|------------|-----------------|
| 工具栏按钮 "📂打开" | 图标风格（线/面）、尺寸（建议 24×24px）、间距（建议 8px） |
| 缩放指示器 "100%▼" | 下拉箭头样式、输入框样式、聚焦态边框颜色 |
| 图片区底色 | 确认色号（UX 建议 #222），是否加纹理/渐变 |
| 全屏浮现工具栏 | 半透明背景透明度（建议 85% 黑）、圆角半径、阴影参数 |
| 状态栏分割线 | 颜色、粗细、是否需要 |
| 错误状态图标 ⚠ | 是否用自定义插画替代 Unicode 字符 |
| 加载动画 ◌ | 旋转环样式、颜色、大小 |
| 右下角缩放浮层 | 是否需要背景、字体大小、渐隐动画 |
