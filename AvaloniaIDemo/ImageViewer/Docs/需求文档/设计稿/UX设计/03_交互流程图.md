## M1 — 打开图片

```
用户操作：双击文件 / 拖拽到窗口 / Ctrl+O / 文件→打开

    ┌─────────────────┐
    │   触发打开请求    │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  检测文件格式     │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │                 │
    ▼                 ▼
 [支持]            [不支持]
    │                 │
    ▼                 ▼
┌─────────┐   ┌─────────────────┐
│检测文件  │   │ 显示错误状态      │
│完整性    │   │ "不支持的格式"    │
└────┬────┘   │ [确定]           │
     │        └────────┬────────┘
 ┌───┼───┐             │
 │       │          用户点击确定
 ▼       ▼             │
[完整] [损坏]           ▼
 │       │       ┌──────────┐
 ▼       ▼       │ 返回空状态 │
┌─────┐ ┌─────┐  │ 或上一张   │
│正常  │ │显示  │  └──────────┘
│解码  │ │损坏  │
│显示  │ │提示  │
└──┬──┘ └──┬──┘
   │       │
   ▼       ▼
┌─────────────────┐
│ 进入浏览状态     │
│ - 激活快捷键     │
│ - 更新状态栏     │
└─────────────────┘
```

### M1 边缘场景

| 场景 | 处理方式 |
|------|---------|
| **GIF 动图** | 正常解码，自动播放（仅循环一次？→ ⚠️ 待需求确认）。旋转对 GIF 是否生效？UX 建议：旋转对 GIF 所有帧生效 |
| **超大图片（>10000×10000）** | 显示加载进度；适应窗口时使用降采样预览；1:1 时允许用户滚动查看 |
| **超长文件名** | 标题栏/状态栏中间截断，Tooltip 显示全名 |
| **慢速存储（U盘/网络盘）** | 显示"加载中..."；超过 10 秒提示"加载较慢，请稍候"，可取消 |
| **同时打开多张** | 以第一张为主视图，其他加入浏览序列；状态栏更新 N/M |
| **拖拽文件夹到窗口** | 遍历文件夹内所有支持的图片格式，以第一张为主视图 |
| **SVG 矢量图** | 缩放不受分辨率限制，但需限制最大缩放（如 6400%） |

## M2 — 缩放

```
用户操作：Ctrl+滚轮 / 点击 +/- / Ctrl+= / Ctrl+- / 快捷键

    ┌─────────────────┐
    │  获取当前缩放比  │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  计算新缩放比    │
    │  （以鼠标位置    │
    │   为缩放中心）   │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │                 │
    ▼                 ▼
 [新比例在范围内]   [新比例超限]
    │                 │
    ▼                 ▼
┌─────────────┐  ┌──────────────┐
│ 应用新缩放比 │  │ 钳制到边界    │
│ 更新状态栏   │  │ (12.5% 或    │
│ 更新指示器   │  │  1600%)      │
└──────┬──────┘  └──────┬───────┘
       │                │
       └───────┬────────┘
               │
               ▼
    ┌─────────────────┐
    │ 重新渲染图片     │
    │ （大图降采样）   │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │ 更新光标         │
    │ （可拖拽→✋）    │
    └─────────────────┘
```

### M2 缩放档位

```
离散缩放档位（按钮 +/-）：
12.5% → 25% → 50% → 75% → 100% → 150% → 200% → 300% → 400% → 800% → 1600%

连续缩放（Ctrl+滚轮）：
平滑缩放，步长约 10%
```

### M2 特殊场景

| 场景 | 处理方式 |
|------|---------|
| **缩放至 100% 以下** | 图片小于窗口时居中，背景显示灰色 |
| **SVG 缩放** | 无上限缩放，但建议最多 6400% |
| **缩放 + 拖拽平移组合** | 放大后鼠标自动变为抓手，拖拽平移不受缩放限制 |
| **适应窗口 (Ctrl+9)** | 等比缩放使图片刚好完整显示在窗口内 |
| **1:1 (Ctrl+0)** | 恢复原始像素大小 |
| **切换图片** | 新图片继承上一张的缩放比例（⚠️ 还是重置为适应窗口？→ UX 建议：重置为适应窗口） |

## M3 — 旋转

```
用户操作：Ctrl+,(逆时针) / Ctrl+.(顺时针) / 工具栏按钮

    ┌─────────────────┐
    │  获取当前旋转角  │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  计算新旋转角    │
    │  (+90° 或 -90°) │
    │  钳制到 0°~270° │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  应用旋转变换    │
    │  （仅视图层）    │
    │  不修改原文件    │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  更新状态栏      │
    │  （旋转角指示）  │
    └─────────────────┘
```

### M3 关键设计决策

| 决策 | 说明 |
|------|------|
| **非破坏性** | 旋转不自动保存，仅在当前会话中生效 |
| **关闭提醒** | 关闭图片/切换图片时若旋转角度 ≠ 0°，弹出 "图片已旋转，是否保存？" [保存] [不保存] [取消] |
| **GIF 动图** | 旋转对所有帧生效（⚠️ 需要在技术验证后确认） |
| **旋转状态指示** | 状态栏中显示旋转图标 + 角度（如 "↻90°"） |
| **双击重置** | 双击旋转按钮可重置旋转角度为 0°（快捷恢复） |

## M4 — 翻看

```
用户操作：← / → / 滚轮（无Ctrl时） / PageUp/PageDown / 工具栏按钮

    ┌─────────────────┐
    │  获取当前索引    │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │        │        │
    ▼        ▼        ▼
 [上一张]  [下一张]  [指定跳转]
    │        │        │
    ▼        ▼        ▼
┌───────┐┌───────┐┌───────────┐
│idx--  ││idx++  ││ idx = 目标│
└──┬────┘└──┬────┘└─────┬─────┘
   │        │            │
   └────────┼────────────┘
            │
            ▼
    ┌─────────────────┐
    │  索引边界检查    │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │                 │
    ▼                 ▼
 [范围有效]       [超出范围]
    │                 │
    ▼                 ▼
┌─────────────┐  ┌──────────────┐
│ 加载新图片   │  │ 循环 / 停止   │
│ 更新状态栏   │  │ (⚠️ 循环模式  │
└─────────────┘  │  待需求确认)  │
                 └──────────────┘
```

### M4 边缘场景

| 场景 | 处理方式 |
|------|---------|
| **翻到不支持格式** | 跳过该文件，继续翻到下一张支持的文件 |
| **翻到损坏文件** | 显示错误状态，用户可选择跳过或关闭 |
| **首张按← / 末张按→** | ⚠️ 循环回第一张/最后一张，还是停止？→ UX 建议：循环 |
| **文件夹只有一张图** | ← 和 → 不响应，无需提示 |
| **GIF 动图翻看** | 翻到 GIF 自动播放，翻走时停止播放 |

## M5 — 全屏

```
用户操作：F11 / F / 双击图片区 / 工具栏全屏按钮

    ┌─────────────────┐
    │  记录当前窗口状态 │
    │  (位置,大小,缩放)│
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  切换到全屏模式   │
    │  - 隐藏标题栏    │
    │  - 工具栏浮动    │
    │  - 背景纯黑      │
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  全屏浏览状态     │
    │  - 图片居中      │
    │  - 工具栏1.5秒   │
    │    后渐隐        │
    │  - 鼠标移动浮现   │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │        │        │
    ▼        ▼        ▼
 [Esc]   [F11/F]   [点击退出按钮]
    │        │        │
    └────────┼────────┘
             │
             ▼
    ┌─────────────────┐
    │  恢复窗口状态     │
    │  (位置,大小,缩放)│
    └─────────────────┘
```

### M5 全屏状态下的操作

| 操作 | 响应 |
|------|------|
| 鼠标移动 | 底部工具栏和状态栏浮现（0.3s 渐显） |
| 鼠标静止 1.5 秒 | 工具栏和状态栏渐隐 |
| 单击图片 | 无操作（⚠️ 或退出全屏？UX 建议：无操作，避免误触） |
| 双击图片 | 退出全屏（与默认模式双击行为保持一致） |
| ← / → | 翻看上一张/下一张 |
| 滚轮（无 Ctrl） | 翻看（⚠️ 或缩放？UX 建议：翻看优先，缩放需 Ctrl+滚轮） |
| Esc | 退出全屏 |

## M6 — 拖拽平移

```
前置条件：图片放大超过窗口（出现滚动条）

用户操作：鼠标拖拽（图片放大状态下）

    ┌─────────────────┐
    │  检测缩放比      │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │                 │
    ▼                 ▼
 [>100%，图片   [≤100%，图片
  超过窗口]      完全可见]
    │                 │
    ▼                 ▼
┌─────────────┐  ┌──────────────┐
│ 光标→✋     │  │ 忽略拖拽      │
│ 可拖拽       │  │ (无操作)     │
└──────┬──────┘  └──────────────┘
       │
       ▼
┌─────────────────┐
│ 鼠标按下         │
│ 光标→✊         │
│ 记录起始位置     │
└────────┬────────┘
       │
       ▼
┌─────────────────┐
│ 鼠标移动         │
│ 计算偏移量       │
│ 更新图片位置     │
└────────┬────────┘
       │
       ▼
┌─────────────────┐
│ 鼠标释放         │
│ 光标恢复→✋     │
│ 图片停在新位置   │
└─────────────────┘
```

### M6 边界处理

| 场景 | 处理方式 |
|------|---------|
| 拖拽到边界 | 超出边界时**弹性回弹**（macOS 风格）或**硬停止**（Windows 风格），取决于平台 |
| 快速甩动 | 惯性滚动（⚠️ 待技术验证可行性） |
| 拖拽时切换图片 | 新图片重置位置（居中） |
