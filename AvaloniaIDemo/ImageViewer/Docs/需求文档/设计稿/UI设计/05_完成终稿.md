| Token | 值 | 用途 |
|---|---|---|
| `--space-xs` | 4px | 紧凑间距（按钮组内间隙、图标与文字间距） |
| `--space-sm` | 8px | 标准间距（工具栏按钮组之间、状态栏信息项之间） |
| `--space-md` | 12px | 宽松间距（工具栏左右内边距、状态栏内边距） |
| `--space-lg` | 16px | 大间距（对话框内边距） |
| `--space-xl` | 24px | 超大间距（空状态内容区上下留白） |

| `--text-xl` | 20px | 28px | Regular | 空状态大图标下方主提示文字 |

### 5.4 图标体系速查

| 属性 | 规范 |
|---|---|
| 风格 | 线性图标（Line） |
| 描边宽度 | 1.5px |

### 5.5 圆角与层次（Elevation）

| 层级 | Token / 表现 | 使用场景 |
|---|---|---|
| Base (0) | `--radius-none` 0px | 工具栏容器、状态栏容器、图片区 |
| 按钮 | `--radius-sm` 4px | 工具栏按钮、下拉项 |
| 浮层 | `--radius-md` 6px | 下拉菜单、缩放指示器、工具提示 |
| 对话框 | `--radius-lg` 8px | 模态面板、对话框 |
| 层次表现 | 亮度差替代阴影：Base `#2C2C2C` → Raised `#353535` + 1px `#454545` → Overlay `#2C2C2C` + 1px `#555` | 下拉/对话框分层 |
| 唯一阴影 | `0px 4px 16px rgba(0,0,0,0.5)` | 仅对话框 |

### 5.6 动效原则

| 原则 | 实现 |
|---|---|
| **功能性动效** | 状态变化时使用（如淡入淡出） |
| **零装饰动效** | 不使用反弹、弹性、视差等"炫技"动效 |
| **时长默认** | 100ms ease（按钮状态切换） |
| **缩放指示器** | 淡出 300ms（采纳 M1 微调） |
| **过渡** | 1px 边框 / 透明度切换，不使用位移/缩放 |

### 5.7 状态栏格式（具体字符与对齐）

```
● 第 N/M 张 │ 文件名 │ 宽×高 │ 大小 │ 格式
   #E8E8E8    #E8E8E8  #999    #999   #999
   12px R     12px R   11px R  11px R 11px R

对齐：左对齐到左内边距 12px
分隔符：` │ `（U+007C + 两侧各 1 空格），颜色 #666
当 M=1 时第一项不显示「第 N/M 张」（按 v1.0 §7 状态栏规则）
放大非 100% 时右对齐追加：150% (16px Semibold #4A90D9)
```

### 5.8 字体回退方案（5.2.1 兼容矩阵）

| 平台 | 字体回退链 |
|---|---|
| **Windows** | "Segoe UI" → "Microsoft YaHei"（无雅黑时）→ system-ui → sans-serif |
| **Linux GNOME / KDE / XFCE** | "Noto Sans" → "Noto Sans CJK SC"（无 Noto 时）→ system-ui → sans-serif |
| **等宽（仅日志/调试）** | "Cascadia Code" → "JetBrains Mono" → "Noto Sans Mono" → monospace |
