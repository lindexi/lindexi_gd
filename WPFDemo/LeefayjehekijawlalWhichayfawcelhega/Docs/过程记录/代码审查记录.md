### 审查结论

本轮实现已经完成从 WPF 空模板到“可生成逐页提示词、支持单页确认图片生成、支持导出已选图片”的基础结构搭建，代码能够成功构建。

### 已确认的设计点

- 使用 `MainViewModel + SlidePageViewModel + Service` 组织主流程。
- 使用 `Microsoft Agent Framework` 调用 `ep-20260306101224-c8mtg` 生成逐页图片提示词。
- 使用独立图片生成服务调用 `ep-20260120102721-c4pxb` 生成候选图片。
- 图片生成前保留用户确认步骤，只有单页点击按钮后才触发图片模型。
- 导出只依赖每页当前已选图片，并按时间命名子目录输出。

### 当前约束

- 豆包 API Key 通过环境变量 `DOUBAO_API_KEY` 读取。
- 图片生成接口按 OpenAI 兼容路径 `/images/generations` 调用。
- 导出目录当前采用手工输入路径的方式预设。
