# 知识库

该目录用于沉淀 AI 智能体在本项目每次协作后的知识总结。

## 强制规则

- 每次 AI 任务结束后，都必须新增一份知识总结文档到本目录。
- 不要覆盖历史总结，按任务维度逐条累积。
- 建议命名：`YYYY-MM-DD-任务简述.md`。

## 建议内容

每份知识总结建议包含以下小节：

- 背景与目标
- 关键文件与模块
- 主要决策与原因
- 修改点摘要
- 验证方式（构建/测试/手工验证）
- 风险与后续建议

## 模板

```markdown
# <任务标题>

## 背景与目标

## 关键文件

- `路径A`
- `路径B`

## 主要决策

## 修改摘要

## 验证记录

## 风险与后续建议
```
