# 02 — 信息架构与工具栏布局

> **文档定位**：定义图片查看器主窗口的信息层级、功能入口布局、工具栏/状态栏方案。  
> **关联 DD 决策**：DD-7（顶部工具栏 + 底部状态栏）、DD-4（状态可见性）  
> **关联 AC**：AC-02～AC-09、AC-11、AC-13

---

## 1. 主窗口信息架构

```
┌──────────────────────────────────────────────────────────────┐
│ ① 标题栏（系统原生 / 自绘）                                    │
│    [应用图标] [文件名 - 图片查看器]      [最小化] [最大化] [关闭] │
├──────────────────────────────────────────────────────────────┤
│ ② 工具栏（常驻，板正克制风格）                                 │
│    [打开] │ [缩放控制组] │ [旋转组] │ [全屏] │ [更多...]        │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                                                              │
│                    ③ 图片显示区域（核心）                       │
│                                                              │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│ ④ 底部状态栏（常驻）                                          │
│    [第 N/M 张]  [文件名.png]  [1920×1080]  [2.3MB]  [150%]  │
└──────────────────────────────────────────────────────────────┘
```

### 层级说明

| 层级 | 名称 | 可见性 | 设计理由 |
|------|------|--------|---------|
| ① | 标题栏 | 常驻（可隐藏于全屏） | 标准桌面窗口惯例，提供窗口管理入口 |
| ② | 工具栏 | 常驻（可隐藏于全屏） | 板正气质：不做"悬浮工具栏"，工具栏就是固定的一部分 |
| ③ | 图片区 | 始终占据最大空间 | 核心任务：看图。任何 UI 元素不能压缩图片区 |
| ④ | 状态栏 | 常驻 | DD-4：缩放比例、位置信息必须始终可见 |

## 2. 工具栏布局设计

### 2.1 工具栏结构

采用**分组式**布局，不是 Ribbon 风格，而是传统的"图标 + 可选文字"工具按钮组。

```
┌─────────────────────────────────────────────────────────────┐
│ [📂打开] │ [-] [+] [100%▼] [1:1] │ [↺左] [↻右] │ [⊞全屏] │ [⋮] │
│   文件    │      缩放控制组        │    旋转组     │   视图   │ 更多│
└─────────────────────────────────────────────────────────────┘
```

### 2.2 各按钮详细设计

#### 文件组（最左）

| 按钮 | 图标建议 | 功能 | 键盘入口 | 工具提示 |
|------|---------|------|---------|---------|
| 打开 | 📂 文件夹图标 | 打开文件对话框 | `Ctrl+O` | "打开图片 (Ctrl+O)" |

> **设计决策**：文件组只放"打开"。不做"另存为/导出"按钮——普通用户不需要。符合"克制"气质。

#### 缩放控制组（居中偏左）

| 按钮 | 图标建议 | 功能 | 键盘入口 | 工具提示 |
|------|---------|------|---------|---------|
| 缩小 | `-` 或 🔍⁻ | 缩小一级 | `Ctrl+-` / `Ctrl+滚轮↓` | "缩小 (Ctrl+-)" |
| 缩放指示器 | 文字 "100%" | 显示当前比例，点击可输入自定义值 | 点击后输入数字+Enter | "当前缩放比例，点击可输入" |
| 放大 | `+` 或 🔍⁺ | 放大一级 | `Ctrl+=` / `Ctrl+滚轮↑` | "放大 (Ctrl+=)" |
| 1:1 | "1:1" 按钮 | 恢复原始大小 | `Ctrl+0` | "原始大小 (Ctrl+0)" |
| 适应窗口 | "适应" 按钮 | 缩放至适应窗口 | `Ctrl+9` | "适应窗口 (Ctrl+9)" |

> **设计决策**：缩放控制组用 `[-] [100%▼] [+]` 三联按钮，这是 macOS Preview 和 Windows 照片都采用的模式，用户零学习成本。`100%▼` 下拉菜单提供 25%/50%/100%/200%/400%/适应窗口 快捷选项。

#### 旋转组

| 按钮 | 图标建议 | 功能 | 键盘入口 | 工具提示 |
|------|---------|------|---------|---------|
| 逆时针旋转 | ↺ | 逆时针 90° | `Ctrl+,` | "逆时针旋转 90° (Ctrl+,)" |
| 顺时针旋转 | ↻ | 顺时针 90° | `Ctrl+.` | "顺时针旋转 90° (Ctrl+.)" |

> **设计决策**：旋转是**非破坏性操作**——旋转状态仅存在于当前会话中，不自动保存（DD-6）。关闭图片时如果有旋转状态，询问是否保存。

#### 视图组

| 按钮 | 图标建议 | 功能 | 键盘入口 | 工具提示 |
|------|---------|------|---------|---------|
| 全屏 | ⊞ | 进入/退出全屏 | `F11` / `F` / `Esc` | "全屏 (F11)" |

#### 更多菜单（最右·三个点）

点击 `⋮` 展开下拉菜单：

```
┌─────────────────────┐
│ 上一张        ←     │
│ 下一张        →     │
│ ─────────────       │
│ 打印...    Ctrl+P   │
│ ─────────────       │
│ 复制图片    Ctrl+C  │
│ 复制到...           │
│ ─────────────       │
│ 属性/详情   Alt+Enter│
└─────────────────────┘
```

> **设计决策**："翻看"不放在常驻工具栏——用户主要通过鼠标滚轮/键盘方向键翻看，不需要两个专用按钮占工具栏空间。但保留在"更多"下拉菜单和键盘入口中。

## 3. 状态栏设计

### 3.1 布局

```
┌──────────────────────────────────────────────────────────────────┐
│ ● 第 3/15 张  │  photo_2024.png  │  1920 × 1080  │  2.3MB  │ 150% │
└──────────────────────────────────────────────────────────────────┘
```

### 3.2 各信息项

| 信息项 | 显示条件 | 格式 | 设计理由 |
|--------|---------|------|---------|
| 图片位置 | 多图浏览时 | "第 N/M 张" | DD-4：位置可见 |
| 文件名 | 始终 | 完整文件名 | 帮助用户确认当前操作对象 |
| 分辨率 | 始终 | "宽 × 高" | 专业用户刚需 |
| 文件大小 | 始终 | 自适应单位（KB/MB） | 普通用户参考 |
| 缩放比例 | 非 100% 时 | "N%" | DD-4：缩放状态可见 |

### 3.3 状态栏行为

- **加载中**：文件名区域显示加载动画 + "加载中..."
- **损坏文件**：分辨率替换为 "⚠ 文件已损坏"
- **不支持格式**：分辨率替换为 "⚠ 不支持的格式"
- **超长文件名**：中间截断，如 "very_long_file_nam...2024.png"，悬停显示完整文件名（Tooltip）

## 4. 图片显示区行为

### 4.1 默认状态

- 图片居中显示
- 背景色：中性灰 `#222`（根据 UI 设计师确认）
- 如果图片小于窗口 → 1:1 显示，居中
- 如果图片大于窗口 → 适应窗口显示

### 4.2 拖拽平移

- 图片放大后，鼠标变为抓手 `✋` 光标
- 拖拽平移时，鼠标按下状态显示 `✊`
- 平移到边界时，弹性回弹或无反应（取决于平台感受）

### 4.3 缩放行为

- 缩放中心：鼠标位置（如鼠标在图片左上角，缩放时以左上角为中心）
- 缩放级别：离散档位：12.5% → 25% → 50% → 75% → 100% → 150% → 200% → 300% → 400% → 800% → 1600%
- 滚轮缩放（Ctrl+滚轮）：平滑缩放，非离散档位

## 5. 全屏模式的信息架构

### 5.1 进入全屏

- 工具栏和状态栏**渐隐**（约 1.5 秒后消失）
- 鼠标移动时工具栏/状态栏**浮现**（约 0.3 秒渐显）
- 底部浮现式工具栏：

```
┌──────────────────────────────────────────────────────────────┐
│                      （图片全屏显示）                          │
│                                                              │
│                                                              │
│         [←] [→] [-] [100%] [+] [↺] [↻] [1:1] [X退出]        │
│          翻看     缩放控制组         旋转     重置    退出     │
└──────────────────────────────────────────────────────────────┘
```

### 5.2 全屏状态栏

- 底部最右侧浮动显示：`"第 3/15 张 | 150%"`
- 不与工具栏冲突

### 5.3 退出全屏

- `Esc` / `F11` / `F` / 点击退出按钮
- 恢复进入前窗口状态（位置、大小）

## 6. 错误状态的信息架构

### 6.1 不支持格式

```
┌─────────────────────────────────────────────┐
│                                             │
│              ⚠                              │
│        无法打开此文件                        │
│    不支持的文件格式：.xyz                    │
│                                             │
│    支持的格式：PNG, JPG, JPEG, BMP,         │
│    GIF, WebP, TIFF, SVG, ICO               │
│                                             │
│           [确定]                            │
│                                             │
└─────────────────────────────────────────────┘
```

### 6.2 损坏文件

```
┌─────────────────────────────────────────────┐
│                                             │
│              ⚠                              │
│        文件已损坏                            │
│    无法读取此图片，文件可能已损坏            │
│    文件名：corrupted_image.png              │
│                                             │
│           [确定]                            │
│                                             │
└─────────────────────────────────────────────┘
```

### 6.3 加载超时（慢速存储）

- 图片区保持中性灰背景
- 居中显示加载动画（转圈 / 进度条）
- 状态栏：文件名 + "加载中..."
- 超过 10 秒无响应 → 显示 "加载时间较长，请稍候…" 并保留 [取消] 按钮

## 7. 与 UI 设计师的协作要点

| 项目 | UX 决策 | UI 细化方向 |
|------|---------|------------|
| 工具栏底色 | 浅灰色（与图片区深灰形成对比） | 具体色号、圆角、分割线样式 |
| 按钮样式 | 图标 + 可选文字 | 图标风格（线/面）、尺寸、间距 |
| 缩放指示器 | 可点击输入 | 输入框样式、聚焦态、下拉菜单样式 |
| 状态栏 | 底部分割线分隔 | 字体、字号、颜色、信息间分割方式 |
| 全屏浮现工具栏 | 底部居中、半透明背景 | 透明度、圆角、阴影 |
| 背景色 | 中性灰 #222 | 确认色号，是否有渐变 |
