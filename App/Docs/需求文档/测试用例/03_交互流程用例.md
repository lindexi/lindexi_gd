## 📌 使用说明

- 本文档覆盖**跨模块的端到端交互流程**，区别于 02 功能模块用例（按模块纵横拆解）与 04 UI 视觉用例（视觉一致性）

- **过渡态用例三段式**：「前后状态 → 当前动作 → 下一状态」
- **异常分支用例三段式**：「触发条件 → 系统反应 → 预期恢复路径」
- 用例标识：`TC-IF-{子模块}-{顺序}-{优先级}`
