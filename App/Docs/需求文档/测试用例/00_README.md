# 测试用例目录

测试工程师产出区。所有测试方案、测试用例、环境就绪报告等文档存放于此。

## 文件清单

- `01_测试方案.md` — 整体测试策略、范围、计划、准入准出标准
- `02_功能模块用例.md` — 各功能模块的测试用例
- `03_交互流程用例.md` — 用户操作流程的端到端测试用例
- `04_UI视觉校验用例.md` — UI 视觉呈现与设计一致性校验用例
- `05_需求-用例映射矩阵.md` — 需求条目与测试用例的对应关系矩阵
- `06_环境就绪报告.md` — 测试环境搭建与就绪确认报告
- `07_P0数据集入库清单.md` — P0 优先级测试数据集的入库与维护清单

## 备注

本目录文件命名采用"序号_文档类型.md"的规范，文档类型聚焦于"产出物形态"而非"测试维度"，便于跨测试类型（如功能、异常、兼容性、性能、易用性等）的内容统一归档与管理。
