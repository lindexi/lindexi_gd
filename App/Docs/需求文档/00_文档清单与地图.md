## 三、术语表

| 术语 | 英文/缩写 | 定义说明 | 出处 |
|------|-----------|---------|------|
| 图片查看器 | Image Viewer | 自研 OS 预装的基础图片查看应用 | R-03 §1.1 |
| 适应窗口 | Fit to Window | 图片等比缩放至完整可见于窗口内 | R-03 §4.1 |
| 视图旋转 | View Rotation | 仅旋转图片在屏幕上的显示，不修改原文件 | R-03 §4.3 |
| 自然排序 | Natural Sort | 按数字逻辑排序（img2 < img10），非字典序 | R-03 §4.4 |
| 多实例 | Multiple Instances | 应用允许同时打开多个独立窗口，且新窗口不替换已有窗口内容 | R-03 §4.1 |
| ImageShape | — | 自研图片解码库，产品依赖的格式支持来源 | R-03 §2.1 |
| Avalonia | — | 跨平台桌面 UI 框架（技术栈） | D-01 §一 |
| AC | Acceptance Criteria | 验收标准，需求的可验证条件 | R-03 |
| DD | Design Decision | 设计决策，UX 设计中的关键决策记录 | UXD-01 |
| VD | Visual Decision | 视觉决策，UI 设计中的关键决策记录 | UID-01 |
| 板正 | — | 产品气质关键词：规矩、正统、专业、克制 | R-03 §1.2 |

## 四、文档结构规范建议

### 4.1 目录命名规范

| 目录 | 命名规范 | 说明 |
|------|---------|------|
| 需求类 | 工作路径根目录 | 以 `图片查看器_需求分析文档_v{版本号}.md` 命名 |
| UX设计 | `设计稿\UX设计\` | 以 `{序号}_{内容}.md` 命名，序号两位数字 |
| UI设计 | `设计稿\UI设计\` | 以 `{序号}_{内容}.md` 命名，序号两位数字 |
| 测试用例 | `测试用例\` | 以 `{序号}_{内容}.md` 命名，序号两位数字 |
| 复审记录 | `复审记录\` | 以 `{角色名}_对{对象}_的复审.md` 命名 |
| 讨论记录 | `讨论记录\` | 以 `{序号}_{阶段描述}.md` 命名 |

### 4.2 文件头模板建议

建议每个文档统一包含以下元信息头：

```markdown
> **文档名称**：{文档全名}
> **文档版本**：v{版本号}
> **创建日期**：{YYYY-MM-DD}
> **最近更新**：{YYYY-MM-DD}
> **责任人**：{角色名}
> **文档状态**：✅ 已完成 / ⬜ 进行中 / 📄 待审核
> **前置文档**：{关联文档列表}
> **阅读对象**：{目标读者}
```
